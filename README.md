# 67-LED-CHASER
![3D View](3d_view.png)
I created this project through a Stasis Hack Club workshop. It consists of nine LEDs, a resistor, a timer IC, an LED IC, a capacitor, and a potentiometer.

## Bill of Materials (BOM)
- Custom PCB board (x1) `You can design this however you want! :)`
- LED (x9) `This depends on your design! (I would recommend 10 or under)`
- NE555P (x1)
- CD4017 (x1)
- Female Header, 1x2 Receptacle (x1)
- _Male/Female Header, 1x1 Receptacle_ (x1) `If you like to debug!`
- 1uF Capacitor (x1)
- 0.01uF Capacitor (x1)
- 1kΩ Resistor (x1)
- 470Ω Resistor (x1)
- Potentiometer (x1)
